Group Project: Optimal Control of Bear Populations in Protected and Harvested Regions: A Metapopulation Approach

This project develops an optimal control strategy for managing black bear populations in a metapopulation system consisting of protected park areas and surrounding harvested forest regions. Using Pontryagin’s Maximum Principle, we formulate and solve an optimal control problem that minimizes human-bear conflicts while maintaining sustainable populations. The model incorporates density-dependent migration patterns and analyzes the effects of different border configurations between protected and harvested areas. Numerical solutions are obtained using a forward-backward sweep method implemented in MATLAB.

Tools: MATLAB

Group Project: Public Transit Timetable Optimization

This project demonstrates the power of Answer Set Programming for solving complex, constrained optimization problems. Translating Graph-Theoretic Clickstream Insights into Tangible UI Design: A Retailrocket Case Study

This study provides a data-driven framework for UI improvements, prioritizing: redesign of critical navigation hubs, reorganization of menu structures to align with detected communities, and targeted interventions for high-exit products. These findings validate that graph-based approaches can effectively diagnose UX friction points and guide strategic interface enhancements in e-commerce platforms. The methodology bridges theoretical network science with practical web design, offering measurable metrics for ongoing UI optimization and personalization efforts..
